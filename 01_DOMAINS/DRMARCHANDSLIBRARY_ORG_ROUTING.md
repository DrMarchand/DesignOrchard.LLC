# DrMarchandsLibrary.org Domain Routing Rule

DrMarchandsLibrary.org organizes data from:

- drmarchandslibrary.com
- drmarchandslaboratory.com

## Source Roles

drmarchandslaboratory.com is a source domain for Laboratory activity, research, builds, workspace output, tests, and system development signals.

drmarchandslibrary.com is a source domain for Library-facing records, references, collections, publications, and knowledge objects.

DrMarchandsLibrary.org is the organizing domain that arranges, reconciles, indexes, and presents those data sources into a clearer Library structure.

## Boundary

DrMarchandsLibrary.org is not the raw Workspace.
DrMarchandsLibrary.org is not the complete Vault.
DrMarchandsLibrary.org is the organizing surface for Library-domain data and Lab-derived records.

## Routing

Laboratory activity may resolve into Library records.
Library-facing records may be organized through DrMarchandsLibrary.org.
Vault-bound records require separate System ID and preservation validation.
