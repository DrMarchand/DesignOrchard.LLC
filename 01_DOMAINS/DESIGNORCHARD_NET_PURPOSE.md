# designorchard.net Purpose Rule

designorchard.net is the network map layer for the Design Orchard ecosystem.

Its purpose is to represent the system as a spiderweb of data with a compass and sundial at its core.

## Core Metaphor

Spiderweb: relationship mesh, data routes, linked objects, communities, companies, homes, workspaces, records, and mirrors.

Compass: directional decision layer that tells the system where to move next.

Sundial: time, sequence, season, lifecycle phase, and historical orientation.

Core: System ID and root reference point used to keep identity stable while data moves.

## Boundary

designorchard.net should not be treated as the LLC itself, the Workspace itself, or the Vault itself.

It is the visible navigation layer that lets the ecosystem see relationships, direction, and time.

## Relationship

Design Orchard LLC is the authority root.
Design Orchard is the ecosystem.
designorchard.net is the network map surface.
Atlas maps topology.
System Compass provides directional intent.
Sundial provides time orientation.
