# Big Brother App Home

The Engine runs between system areas. Before the Engine becomes remote, it needs a defined origin point.

## Domain Roles

- app.drmarchandslaboratory.com = Big Brother app home
- Big Brother = application surface connected to Engine routing
- Engine = routing layer between system areas
- Remote Engine = allowed only after origin identity is defined

## Boundary

Big Brother is not the full Engine.
Big Brother is not the OS Home.
Big Brother is not the LLC root.

## Rule

Origin first, then remote.

The Engine should not become remote until Big Brother has a stable home, System ID alignment, authority boundary, and Atlas route.
