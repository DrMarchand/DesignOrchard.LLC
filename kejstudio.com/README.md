# kejstudio.com

kejstudio.com is the domain package for KEJ Studio inside the Design Orchard LLC repository.

## Repository Path

DrMarchand/DesignOrchard.LLC/kejstudio.com/

## Role

kejstudio.com supports the KEJ Studio public/domain identity and acts as a domain-home folder for documentation, routing notes, manifests, schemas, creative production references, and deployment references.

## Boundary

This repository path is not the public website itself.
It is the structured source and documentation location for the kejstudio.com domain package.

## Rule

kejstudio.com domain artifacts should resolve through this path before being promoted, mirrored, deployed, archived, or connected to other ecosystem domains.
