# DrMarchands.com

DrMarchands.com is the domain package for the DrMarchand professional and software identity inside the Design Orchard LLC repository.

## Repository Path

DrMarchand/DesignOrchard.LLC/DrMarchands.com

## Role

DrMarchands.com supports the DrMarchand professional/software identity and acts as a domain-home folder for documentation, routing notes, manifests, schemas, and deployment references.

## Boundary

This repository path is not the public website itself.
It is the structured source and documentation location for the DrMarchands.com domain package.

## Rule

DrMarchands.com domain artifacts should resolve through this path before being promoted, mirrored, deployed, archived, or connected to other ecosystem domains.
