# Nested Domain Package

Repository path:

DrMarchand/DesignOrchard.LLC/DrMarchands.com/DrMarchandsLibrary.org/DrMarchandsLaboratory.com

## Purpose

This folder records a nested domain relationship.

DrMarchands.com is the parent identity context.
DrMarchandsLibrary.org is the library organizing context.
DrMarchandsLaboratory.com is the laboratory domain context.

## Boundary

This folder documents a relationship path.
It does not replace standalone domain folders.

## Rule

Use this folder for relationship notes, manifests, and cross-domain context.
