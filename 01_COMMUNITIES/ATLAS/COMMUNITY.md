## Definition

DrMarchand Atlas is the shared operating map, topology layer, and coordination surface used by the operating system and the execution engine.

Atlas maps the ecosystem so the operating system can govern it and the engine can execute it.

## Placement

Atlas exists inside the operating system, inside the engine, and between them as the shared map layer.
