## Purpose

This map separates domains, websites, dashboards, entities, bridges, engines, intelligence layers, and reporting layers before registry certification.

## Core distinction

A domain is not a website.
A website is not a system.
A system is not a database.
A database is not an asset.
A bridge is not an entity.

Each object must receive its own registry classification before certification.
