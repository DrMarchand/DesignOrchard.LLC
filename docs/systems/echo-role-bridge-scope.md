# Echo Role Bridge Scope

Date: 2026-06-23

## Definition

Echo is a Discord role bridge bot for The Creative Guild.

Echo is not:

- an entity
- a product
- a website
- a dashboard
- a certification authority

## Phase 1 scope

Echo should stay small during Phase 1.

Core duties:

- Role change requests
- Eligibility checks
- Approval logging
- Permission boundary protection
- Status reporting back to Big Brother / DrMarchand

## Out of scope for Phase 1

Echo should not handle:

- payments
- dashboards
- asset approval
- registry certification
- public website behavior
- Neuro-Forge Engine™ control beyond reporting

## Relationship

The Creative Guild is a Discord bridge only. Echo supports that bridge by managing controlled role changes and role-change reporting.
