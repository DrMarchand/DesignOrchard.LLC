# DesignOrchard.net Route Manifest

This file records a network route for DesignOrchard.net.

## Route

DesignOrchard.net / DrMarchand / DesignOrchard.LLC / DrMarchands.com / DrMarchandsLibrary.org / DrMarchandsLaboratory.com

## Meaning

DesignOrchard.net is the map surface.
DrMarchand is the namespace context.
DesignOrchard.LLC is the LLC and repository context.
DrMarchands.com is the identity domain context.
DrMarchandsLibrary.org is the Library organizing context.
DrMarchandsLaboratory.com is the Laboratory domain context.

## Boundary

This route documents a relationship chain.
